# STM32 Switch Controls LED

## Overview
This project demonstrates digital input and output by controlling an LED using a push button.

## Features
- GPIO Input
- GPIO Output
- Push Button Interface
- LED Control

## Hardware
- STM32 NUCLEO-F446RE
- Push Button
- LED

## Working
The microcontroller continuously reads the button state.
- Button Pressed → LED ON
- Button Released → LED OFF

## Concepts Covered
- GPIO Input
- GPIO Output
- Digital Read
- Polling Method

## Software
- STM32CubeIDE
- STM32 HAL Driver
