# STM32 PWM LED Brightness Control

## Overview
This project demonstrates LED brightness control using PWM.

## Features
- PWM Generation
- Variable Duty Cycle
- Brightness Control

## Hardware
- STM32 NUCLEO-F446RE
- LED

## Working
Changing the PWM duty cycle changes the LED brightness.

## Concepts Covered
- PWM
- Timer
- Duty Cycle
