# STM32 Multiple LED Blink

## Overview
This project demonstrates blinking multiple LEDs with different timing patterns.

## Features
- Multiple GPIO Outputs
- LED Pattern Generation
- Timing Control

## Hardware
- STM32 NUCLEO-F446RE
- Multiple LEDs

## Working
Each LED blinks in a predefined sequence to demonstrate GPIO control and timing.

## Concepts Covered
- GPIO
- Delay
- LED Pattern Control

## Software
- STM32CubeIDE
