# STM32 One Switch Multiple LED Control

## Overview
This project demonstrates controlling multiple LEDs using a single push button.

## Features
- Single Button Interface
- Multiple LED Control
- GPIO Programming

## Hardware
- STM32 NUCLEO-F446RE
- Push Button
- Multiple LEDs

## Working
When the push button is pressed, multiple LEDs change their states according to the programmed sequence.

## Concepts Covered
- GPIO Input
- GPIO Output
- Multiple Device Control

## Software
- STM32CubeIDE
- STM32 HAL Driver
