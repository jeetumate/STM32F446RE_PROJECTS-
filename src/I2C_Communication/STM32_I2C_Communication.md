# STM32 I2C Communication

## Overview
This project demonstrates I2C communication between STM32 and an external I2C device.

## Features
- I2C Master
- Data Transmission
- Device Communication

## Hardware
- STM32 NUCLEO-F446RE
- I2C Peripheral (LCD/OLED/Sensor)

## Concepts Covered
- I2C Protocol
- Master Communication
- HAL I2C Driver
