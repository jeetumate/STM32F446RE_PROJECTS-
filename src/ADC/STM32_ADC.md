# STM32 ADC Example

## Overview
This project demonstrates Analog-to-Digital Conversion using STM32.

## Features
- ADC Configuration
- Analog Input Reading
- Voltage Measurement

## Hardware
- STM32 NUCLEO-F446RE
- Potentiometer

## Working
The ADC continuously samples the analog voltage from the potentiometer and converts it into a digital value.

## Concepts Covered
- ADC
- Analog Input
- Voltage Conversion

## Software
- STM32CubeIDE
- STM32 HAL Driver
