# STM32 Embedded Systems Projects

A collection of Embedded Systems projects developed using the **STM32 NUCLEO-F446RE** development board and **STM32CubeIDE**. This repository contains beginner to intermediate examples covering GPIO, Interrupts, Communication Protocols, ADC, PWM, RTOS, LoRa communication, and mini projects.

---

## 📖 About

This repository was created as part of my Embedded Systems learning journey. Each project demonstrates a specific STM32 peripheral or embedded concept using the STM32 HAL library.

The projects are designed to help students and beginners understand how to interface hardware with STM32 microcontrollers and build real-time embedded applications.

# 📂 Projects Included

| Project | Description |
|----------|-------------|
| 1_Led_Blink | Basic LED blinking using GPIO output. |
| Builtin_led_Blink | Controls the onboard LED (LD2) of the STM32 NUCLEO board. |
| 3_LED_BLINK | Demonstrates blinking multiple LEDs in different patterns. |
| 1_Switch_1_LED | Controls an LED using a push button. |
| 1_Switch_Multiple_LED_Control | Controls multiple LEDs using a single switch. |
| External_interrupt | Demonstrates GPIO External Interrupt (EXTI). |
| Timer_Interrupt_ | Uses hardware timers to generate periodic interrupts. |
| UART_Communication | Serial communication between STM32 and PC using UART. |
| I2C_Communication | Demonstrates I2C communication with external devices. |
| ADC | Reads analog signals using the ADC peripheral. |
| PWM_LED | Controls LED brightness using PWM. |
| PWM_Motor | Controls DC motor speed using PWM. |
| Temp_Monitor | Reads and monitors temperature sensor data. |
| Using_RTOS_MiniProject | Demonstrates multitasking using FreeRTOS on STM32. |
| Wireless_Lora_Communication | Wireless communication using LoRa modules. |
| KWTMS | Complete embedded application demonstrating multiple STM32 peripherals and real-world implementation. |
